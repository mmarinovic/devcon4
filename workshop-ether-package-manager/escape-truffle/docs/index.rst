
Ethereum Package Registry (V2)
==============================

Install
-------

.. code-block:: shell

   $ npm install
   $ docker pull ethereum/client-go:v1.8.6


Test
----
.. code-block:: shell

   $ npm test        # vs. ganache-cli
   $ npm test:geth   # vs. geth


Coverage
--------

.. code-block:: shell

   $ npm run coverage


Lint
----
For help, see the `Solium documentation <https://github.com/duaraghav8/Solium>`.

.. code-block:: shell

   $ npm run lint
